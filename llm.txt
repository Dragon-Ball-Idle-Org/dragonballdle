# DragonBallDle

> DragonBallDle is a daily guessing game for Dragon Ball fans, inspired by Wordle. Test your knowledge with different game modes like Classic and Silhouette. It is a non-commercial fan project.

## About

- Type: Web application (Next.js), multilingual UI
- Locales: en-US (default), pt-BR, and 18+ other languages.
- Status: Fan-made, non-commercial project

## Entry points

- [Home — English](https://dragonballdle.com/en-US)
- [Home — Portuguese](https://dragonballdle.com/pt-BR)

## Game Modes

- [Classic Mode — EN](https://dragonballdle.com/en-US/classic)
